# fireworks-canvas

🎆 Click or tap anywhere to launch fireworks.

Canvas と JavaScript だけで描いた、  
夜空に広がるデジタル花火です。

スマートフォンでもスムーズに動作し、  
画面いっぱいに広がる粒子と余韻を楽しめます。

※ 本リポジトリは JavaScript / Canvas の習作です。

## Demo
https://hiro-kuroe.github.io/fireworks-canvas/

## How to use
- クリック / タップで花火を打ち上げ  
- PC / スマートフォン対応  

## Tech
- HTML  
- JavaScript  
- Canvas API
